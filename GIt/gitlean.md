# GIT常用语句 
### 构建仓库
``` git init ```
### 添加文件
##### 添加文件到暂存
``` git add [file1] [file2] ... ```

##### 添加指定目录到暂存区，包括子目录：
``` git add [dir] ```
##### 添加当前目录下的所有文件到暂存区：
``` git add . ```
##### 用于查看项目的当前状态
``` git status ```
